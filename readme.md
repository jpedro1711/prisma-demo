### Comandos importantes

### Start aplicação
#### npx ts-node src/index.ts 

### Criar migração
#### npx prisma migrate dev --name addCreatedAtToUser

### Instalar prisma
#### npm i prisma --save-dev

### Gerar prisma client se não existe
#### npx prisma generate

### Iniciar prisma client
#### npx prisma init

### Iniciar banco de dados
#### docker-compose up -d
